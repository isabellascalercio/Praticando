Questionário

Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: A função de um servidor web em um ambiente de hospedagem é receber, processar e responder às requisições dos usuários feitas por meio de navegadores, entregando páginas,arquivos ou dados solicitados. Ele atua como intermediário entre o usuário e a aplicação hospedada.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: O NGINX é leve e rápido. Ele consegue lidar com milhares de conexões ao mesmo tempo, consumindo poucos recursos. Por isso, é muito usado em sites com muito tráfego.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP envia dados sem segurança. Já HTTPS criptografa os dados, protegendo contra espionagem e ataques. HTTPS é recomendado para proteger senhas, dados pessoais e garantir privacidade.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: Portas identificam qual serviço ou aplicação deve receber os dados que chegam. A porta 443 é usada para conexões HTTPS, que são seguras.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: O IP é como o “endereço” de um dispositivo na rede. Ele permite que computadores se encontrem e se comuniquem.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: VPS é uma hospedagem virtual com recursos dedicados (RAM, CPU). É ideal para sites médios ou com mais tráfego que precisam de controle e desempenho maior que uma hospedagem compartilhada.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: O Caddy é fácil de configurar e já vem com HTTPS automático. É moderno e seguro, ideal para quem quer simplicidade.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: O IIS (Internet Information Services) é o mais indicado, pois é feito pela Microsoft e se integra perfeitamente com o .NET.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: O cliente (navegador) faz um pedido e o servidor responde com os dados (como páginas HTML, arquivos etc.).
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: O Tomcat é um servidor de aplicação para rodar aplicações Java, especialmente as que usam servlets e JSP.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Servidor web entrega arquivos estáticos (HTML, imagens). Servidor de aplicação executa código e regras de negócio, como login e banco de dados.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: O domínio (como google.com) é um nome fácil de lembrar que aponta para um IP. Ele evita que precisemos decorar números.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

Resposta: Escalabilidade, alta disponibilidade, menor risco de falhas físicas e acesso remoto de qualquer lugar.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

Resposta: Tomcat para Java e Gunicorn para Python.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

Resposta: HTTPS protege contra roubo de dados. HTTP não criptografa as informações, o que permite ataques como interceptação de senhas.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

Resposta: Para uma loja virtual com muitos acessos diários, pois o VPS oferece mais controle e desempenho.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

Resposta: IP identifica o computador. Porta identifica o serviço dentro do computador (como web ou e-mail).
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Resposta: Garante segurança e confiança dos usuários, além de ser exigido por navegadores modernos e mecanismos de busca.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

Resposta: NGINX é leve e de alto desempenho. Apache HTTP Server é flexível e fácil de configurar.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

Resposta: Vários sites compartilham o mesmo servidor. É barata e ideal para sites pequenos ou pessoais.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

Resposta: Ele processa lógica do sistema, acessa banco de dados e executa código. Servidores estáticos só entregam arquivos prontos.
Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

Resposta: Gunicorn recebe requisições e distribui entre vários processos Python para rodar aplicações web como Flask ou Django.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

Resposta: É um servidor de aplicação Java para rodar sistemas corporativos complexos (Java EE/Jakarta EE).
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

Resposta: É usada para FTP, protocolo de transferência de arquivos.
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Resposta: Pode usar as mesmas configurações e plugins do Apache, mas com mais desempenho e segurança.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

Resposta: É o servidor web da Microsoft, usado em sistemas com .NET e Windows, especialmente em empresas que usam tecnologias Microsoft.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

Resposta: Permite rodar JavaScript no servidor. É usado em APIs, aplicações em tempo real e sistemas modernos.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

Resposta: Roda aplicações Java EE (Jakarta EE). Usado em aplicações empresariais Java.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

Resposta: São servidores para aplicações Ruby (como Ruby on Rails), que lidam com múltiplas conexões de forma eficiente.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

Resposta: Afeta desempenho, custo, escalabilidade e segurança. A escolha errada pode causar lentidão, instabilidade ou gastos desnecessários.
